# Webcam-Object-Tracking-System
Real-time object tracking system using OpenCV and Python. Users can select objects manually and track them through live webcam video.
## Overview

A computer vision project that tracks user-selected objects in real time using a webcam and OpenCV.

The user captures a frame from the webcam and selects one or more objects by drawing bounding boxes. The system then creates trackers and continuously follows the selected objects in the live video stream.

## Features

* Real-time webcam tracking
* Manual object selection
* Multiple object tracking
* CSRT tracking algorithm
* Live bounding box updates

## Technologies

* Python
* OpenCV
* Jupyter Notebook

## Applications

* Surveillance systems
* Human motion tracking
* Traffic monitoring
* Computer vision research
